# Binance Futures Trading Bot

## Features

- BUY Orders
- SELL Orders
- MARKET Orders
- LIMIT Orders
- Input Validation
- Logging
- CLI Interface

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python cli.py
```

## Example

Symbol: BTCUSDT

Side: BUY

Order Type: MARKET

Quantity: 0.001
